Basic lexer that supports:

- Integer (int)
- String (string)
- Integer literals (e.g. 1, 2, 3, etc.)
- String literals (e.g. "Hello", "World", "123", etc.)
- Parentheses ((, ))
- Braces ({, })
- Boolean operators (==, ||, &&, <, >, <=, >=)
- Assigment operator (=)
- Basic control statements (if/else/for/while)
- [Inc/Dec]rement (++, --)
- Semi-colons
